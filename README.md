# ios-from-web-guide

A Claude Code plugin that enforces best practices when adding a native iOS client to an existing web application. Targets SwiftUI, iOS 17+, XcodeGen project generation, and the App Store release pipeline. Pairs enforced skills with hooks and templates so common mistakes are caught before they ship.

## Status

Pre-release — 1.0.0 is under active development.
